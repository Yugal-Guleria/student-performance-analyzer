# Student Performance Analyzer

A simple Python project that analyzes student marks and gives clear subject-wise insights along with useful suggestions.

## Features

* Works for Class 10th and 12th
* Calculates total marks, percentage, and grade
* Shows subject-wise performance (Weak / Average / Strong)
* Gives simple and personalized improvement advice

## How It Works

1. The user selects the class (10th or 12th)
2. Enters marks for each subject
3. The program calculates:

   * Total marks
   * Percentage
   * Grade
4. It then shows subject-wise feedback and overall advice

## How to Run

1. Install Python on your system
2. Run the file using:
   python filename.py
